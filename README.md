## üåü How to Use

After installation, navigate to any Amazon product page. Look for the new copy button added by AmazonProductCopier.

1. **Locate the Copy Button**: It should appear on the product page, usually near the product details.
2. **Click the Button**: This action copies all relevant product information to your clipboard.
3. **Paste Anywhere**: Simply go to your preferred document or note-taking application and paste the information using `CTRL + V` (Windows) or `CMD + V` (Mac).
